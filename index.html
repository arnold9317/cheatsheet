HTML Table Reference

Table Tags

Tag Name Description
<form> Form Used to collect and send information to an external source.
<input> Input Used for user input in forms, supporting various types.
<textarea> Textarea Used when creating a text-box for multi-line input.
<select> Select Used for creating a dropdown list.
<datalist> Datalist When using an HTML input, a basic search/autocomplete functionality can be achieved by pairing it with a <datalist>.
<label> Label It associates text labels with form elements for improved accessibility.

Table attributes

Attribute Name Description
name Name The "name" attribute in HTML assigns a name to an element, to identify data when submitted.
required Required Specifies that the field must include a value.
maxlength Maximum Length Sets the maximum length for input elements, enforcing a maximum character count.
minlength Minimum Length Sets the minimum length for input elements, enforcing a minimum character count.
max Max Specifies the maximum value that can be entered into the field, in number type forms.
min Min Specifies the minimum value that can be entered into the field, in number type forms.
pattern Pattern Sets a regular expression pattern for input validation, ensuring a specific format is followed.
